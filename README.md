# Electric-Vehicle-Adoption-Dashboard

📊 Key Metrics & KPIs

Total Vehicle Count: 247K+ registrations tracked.

Year-Over-Year (YOY) Growth %: Measures the velocity of market acceleration.

Average Electric Range (44.65 miles): Highlights the shift toward high-volume commuter models.

Top 10 Rankings: Performance breakdown by City, County, and Model.

💡 Key Insights

Tesla’s Market Dominance: The Model Y and Model 3 significantly outpace competitors, indicating strong brand loyalty.

Regional Hotspots: King County (Seattle area) serves as the primary hub for adoption and a blueprint for infrastructure needs.

The Range Paradox: While early luxury models reached higher peak ranges, market volume is now driven by mid-range, high-efficiency models.

🛠️ Tech Stack & Tools

Power BI: Primary tool for data visualization and report generation.

DAX (Data Analysis Expressions): Used for advanced time-intelligence measures (YOY Growth).

Power Query: For ETL (Extract, Transform, Load) and data cleaning.

Data Modeling: Integrated reporting using a star schema or relational model.
